Application Management Application

The purpose of this application is to manage appointments and customer information. 

Author: Long Tran
Contact: ltran56@my.wgu.edu
Version: 1.0.0
Date: March 28, 2022

Made with: 
IntelliJ IDEA 2022.3 (Community Edition), 
Java SE 11.0.11, 
JavaFX SDK 17.0.1,
mysql-connector-java-8.0.25

How to run program:
1. Unzip "C195 (Software II) - Long Tran.zip" and browse to "C195 (Software II) - Long Tran\src\sample" folder containing .java files. 
2. Open "sample" folder in IntelliJ.
3. Build and run project from Main (public static void main(String[] args) throws SQLException {...})
4. Enter username and password and press "Log In" to access appointment management screen.

Reports Screen: The "View Reports" button will open a window showing the number of appointments by type and month. An additional report tracks the user's recent activities involving adding, updating and deleting appointments and customer information and the date and time of the activities.
